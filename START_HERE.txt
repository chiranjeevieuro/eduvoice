## 🚀 HOW TO RUN

### SIMPLEST: Run Demo (30 seconds, no setup)
```bash
cd "THE PARK APL-VOICE FIRST APP"
python demo_simple.py
```
**Result**: See all 4 scenarios (Student, Faculty, Admin, Multilingual)

### QUICK: Run Full App (Text Mode)
```bash
python main.py --user-type student --mode text
```
Then interact with natural language commands.

### OPTIONS:
```bash
# Faculty mode
python main.py --user-type faculty --mode text

# Admin mode
python main.py --user-type admin --mode text

# Voice mode (optional)
python main.py --user-type student --mode voice

# Demo with detailed output
python demo.py
```
